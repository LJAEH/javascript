변수선언방식

        Javascript는 변수 선언시 자료형이 아닌 
        var , let, const, (미작성) 중 하나를 골라서 사용함
        
        var : 변수, 변수명 중복 가능 (덮어쓰기)
        let : 변수, 변수명 중복 불가 블록 레벨 scope
        const : 상수, 변수명 중복x 블록{} scope

        1순위 : const (JS 는 HTML상에서 )

Javascript의 자료형

        JS는 변수 선언시 별도의 자료형을 지정하지 않음

        => 변수에 대입되는 값(== 리터럴)에 의해서 자료형이 결정됨

        - string (문자열/"abc",'abc')
        - number (숫자 / 100(정수), -3.14(실수))
        - boolean (논리값 / true , false)
        - object (객체 / 배열(Array), JS객체 {K:V}, {K,V})
        - function (함수)
        - undifined (자료형이 정의되지않은 변수)

        * null 은 참조하는게 없음을 나타내는 리터럴(자료형X)
